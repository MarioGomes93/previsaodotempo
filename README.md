# WeatherWise - Previsão do Tempo

Aplicativo simples de consulta da situação do clima em várias partes do mundo usando HTML, JavaScript e CSS. Consome a API do [OpenWeatherMap](https://openweathermap.org/).

## Funcionalidades

- Busca de previsão do tempo por cidade
- Exibe temperatura, condição climática e umidade
- Interface moderna e responsiva
- Ícones do clima dinâmicos

## Como usar

1. Abra o arquivo `index.html` em um navegador web.
2. Digite o nome da cidade no campo de entrada.
3. Clique em "Buscar Previsão" para ver os dados meteorológicos.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- API do OpenWeatherMap

## Configuração da API

A aplicação utiliza uma chave da API do OpenWeatherMap. Para obter sua própria chave:

1. Acesse [OpenWeatherMap](https://openweathermap.org/api) e crie uma conta.
2. Gere uma chave de API gratuita.
3. Substitua a chave no arquivo `script.js` (linha 1).

## Estrutura do projeto

- `index.html`: Estrutura da página
- `styles.css`: Estilos da interface
- `script.js`: Lógica da aplicação
